HTML
Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages.
[img src="image.jpeg" alt="image.com]
href attributes
The [a] tag defines a hyperlink. The href attributes specifies the URL of the link.
*[a href="https//www.w3schools.com"] Visit w3schools.com
Start Tag, Element Content and End Tag
An HTML element is defined by a starting tag. If the element contains other content, it ends with a closing tag.
[p][/p][/body][body]
DOCTYPE html Declaration
A document type declaration, or DOCTYPE, is an instruction that associates a particular XML or SGML document with a document type definition.
[!DOCTYPE html]
HTML Element
An HTML element is a type of HTML document component, one of several types of HTML nodes.
[ul][/ul][ol][/ol][li][/li]
Head Element
machine-readable information (metadata) about the document, like its title, scripts, and style sheets.
[html][body][head]
Title Element
Defines the document's title that is shown in a browser's title bar or a page's tab. It only contains text; tags within the element are ignored.
[title]
Body Element
Contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
[body]
HTML Headings
A heading element implies all the font changes, paragraph breaks before and after, and any white space necessary to render the heading.
[h1][h2][h3]
HTML Paragraph
A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.
[p]
HTML Links
HTML links are hyperlinks. You can click on a link and jump to another document. When you move the mouse over a link, the mouse arrow will turn into a little hand.
[a href="url">link text[/a]
HTML Image
Is used to embed an image in a web page.
[img]
HTML Element
The HTML element is everything from the start tag to the end tag:
[h1]My First Heading[/h1]
Nested HTML Elements
A nested element, also called a child element, can be a parent element too if other elements are nested within it.
[em][/em]
HTML Attributes
HTML attributes are special words used inside the opening tag to control the element's behaviour. HTML attributes are a modifier of an HTML element type.
[img src="img_girl.jpg"]
  • Href attributes
  • Src Attributes
  • Width and Height Attributes
  • Style Attributes
  • Alt Attributes
HTML Horizontal Rules
Used to insert a horizontal rule or a thematic break in an HTML page to divide or separate document sections.
[hr style="width:50%;text-align:left;margin-left:0"]
HTML Line Breaks
The HTML Line Breaks tag inserts a single line break.
[p]To force[br] line breaks[br] in a text,[br] use the br[br] element.[/p]
HTML Style Attributes
The style attribute specifies an inline style for an element.
[p] style="color:green">This is a paragraph.[/p]
  • Background Color
  • Font
  • Text Size
  • Text Color
  • Text Allignment
HTML Formatting Elements
Character formatting includes selecting a font, a font size, bold or italics, and so on. At the paragraph level, you apply indents, bullets, and line spacing.
[b]This text is bold[/b]
  • Important Text
  • Bold Text
